# CHZZK Image Spoiler Blocker

치지직(`https://chzzk.naver.com/`)에서 이미지와 배경 이미지를 숨기는 Chrome 확장입니다.

## 설치

1. Chrome에서 `chrome://extensions`를 엽니다.
2. 우측 상단의 `개발자 모드`를 켭니다.
3. `압축해제된 확장 프로그램을 로드`를 누릅니다.
4. 이 폴더(`C:\www\chrome_extensions\chzzk`)를 선택합니다.

## 사용

확장 아이콘을 눌러 이미지 숨김을 켜거나 끌 수 있습니다.
기본값은 `숨김 켜짐`입니다.
